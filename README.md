# Przepisy (Express + SQLite)

Prosta aplikacja Express do przechowywania filmów w bazie SQLite.

## Wymagania
- Node.js 18+  
- npm

## Instalacja
1. Zainstaluj zależności:
   npm install


## Uruchamianie
1. Uruchom serwer:
   npm start
   lub: node index.js

2. Otwórz w przeglądarce:
   http://localhost:8000/
